# 客户管理系统 - 简化版需求

## 一、核心功能范围

基于Excel工作表"使用说明"和"General-Fishpool"，实现两个核心模块：

### 1. Fishpool（客户池管理）
- 客户基本信息管理
- 客户三维评分（X/Y/Z）
- 客户分级（A/B/C/D）
- 客户列表和筛选

### 2. 客户Action（客户状态流转）
基于"使用说明"中定义的客户开发流程：
- new opportunity（新商机）
- New Inquire（新询价）
- PO（订单）
- follow（跟进）
- Customer Feedback（客户反馈）
- Close（关闭）

## 二、数据库设计

### 2.1 客户表 (customers)

```sql
CREATE TABLE customers (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    
    -- 基本信息
    client_name VARCHAR(200) NOT NULL COMMENT '客户名称',
    business_model VARCHAR(20) COMMENT '业务模式: Farming/Hunting',
    area VARCHAR(100) COMMENT '区域',
    address TEXT COMMENT '地址',
    
    -- 三维评分
    description_x TEXT COMMENT '客户描述（X轴）',
    score_x INT DEFAULT 0 COMMENT '客户潜力评分 0-100',
    
    description_y TEXT COMMENT '竞争环境描述（Y轴）',
    score_y INT DEFAULT 0 COMMENT '竞争评分 0-100',
    
    key_person VARCHAR(200) COMMENT '关键联系人（Z轴）',
    score_z INT DEFAULT 0 COMMENT '关系评分 0-100',
    
    -- 分级和战略
    level VARCHAR(1) COMMENT '客户等级: A/B/C/D',
    client_strategy TEXT COMMENT '客户策略',
    potential_contribution DECIMAL(10,2) COMMENT '年度潜在贡献(KEUR)',
    
    -- 当前状态
    current_stage VARCHAR(50) DEFAULT 'new_opportunity' COMMENT '当前阶段',
    
    remark TEXT COMMENT '备注',
    status VARCHAR(20) DEFAULT 'active' COMMENT '状态: active/inactive',
    
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    updated_at DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
    
    INDEX idx_level (level),
    INDEX idx_area (area),
    INDEX idx_stage (current_stage),
    INDEX idx_business_model (business_model)
) COMMENT='客户信息表';
```

### 2.2 客户动作记录表 (customer_actions)

```sql
CREATE TABLE customer_actions (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    customer_id BIGINT NOT NULL COMMENT '客户ID',
    
    -- 动作信息
    action_type VARCHAR(50) NOT NULL COMMENT '动作类型: new_opportunity/new_inquire/po/follow/customer_feedback/close',
    action_date DATE NOT NULL COMMENT '动作日期',
    
    -- 动作内容
    title VARCHAR(200) COMMENT '标题',
    content TEXT COMMENT '内容描述',
    result TEXT COMMENT '结果',
    
    -- 附加信息
    estimated_value DECIMAL(12,2) COMMENT '预估价值(EUR)',
    next_action TEXT COMMENT '下一步行动',
    
    created_by VARCHAR(100) COMMENT '创建人',
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    updated_at DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
    
    FOREIGN KEY (customer_id) REFERENCES customers(id) ON DELETE CASCADE,
    INDEX idx_customer (customer_id),
    INDEX idx_type (action_type),
    INDEX idx_date (action_date)
) COMMENT='客户动作记录表';
```

### 2.3 联系人表 (contacts)

```sql
CREATE TABLE contacts (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    customer_id BIGINT NOT NULL COMMENT '客户ID',
    
    name VARCHAR(100) NOT NULL COMMENT '姓名',
    position VARCHAR(100) COMMENT '职位',
    phone VARCHAR(50) COMMENT '电话',
    email VARCHAR(100) COMMENT '邮箱',
    is_key_person BOOLEAN DEFAULT FALSE COMMENT '是否关键人',
    
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    updated_at DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
    
    FOREIGN KEY (customer_id) REFERENCES customers(id) ON DELETE CASCADE,
    INDEX idx_customer (customer_id)
) COMMENT='联系人表';
```

## 六、客户等级计算规则

```javascript
function calculateLevel(scoreX, scoreY, scoreZ) {
  // Level A: 三项都高 (>=70)
  if (scoreX >= 70 && scoreY >= 70 && scoreZ >= 70) {
    return 'A';
  }
  
  // Level B: 至少两项高
  const highScores = [scoreX >= 70, scoreY >= 70, scoreZ >= 70].filter(Boolean).length;
  if (highScores >= 2) {
    return 'B';
  }
  
  // Level C: 客户潜力高，但其他项低
  if (scoreX >= 70) {
    return 'C';
  }
  
  // Level D: 其他情况
  return 'D';
}
```
